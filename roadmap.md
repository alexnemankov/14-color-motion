# Product Roadmap

This roadmap is intentionally future-facing. It only tracks remaining work that is still useful for development planning.

## Direction

The product should keep moving toward a stronger creative tool for procedural color motion, with emphasis on:

- faster experimentation
- better output formats
- deeper ambient/display use
- cleaner long-term product identity

## Active Work

### Phase 2: Creative Workflow

Goal: make experimentation faster, safer, and more satisfying.

Remaining:

- Add A/B compare workflow
- Make saved-result exploration feel less disposable inside the workflow itself

### Phase 4: Shareability and Output

Goal: turn scenes into reusable artifacts.

Remaining:

- Add GIF export if the quality/performance tradeoff is acceptable
- Add more export presets or recording controls when there is a clear user need

### Phase 5: Interactivity and Depth

Goal: deepen the experience for advanced users.

Remaining:

- Add pointer/touch interaction in more renderers:
  - blobs deform around touch
  - turing injects pigment on click
  - waves distort from pointer
  - more deliberate touch behavior for mobile
- Add presentation mode:
  - auto-hide UI
  - cycle saved presets
  - timed transitions
- Add performance modes:
  - quality
  - balanced
  - performance

## Optional Follow-Up

These are not immediate priorities, but still worth keeping in view:

- deeper palette grouping or editorial collections if the library grows
- further branding refinement if the product name or voice changes
- extra mobile polish if the panel still feels dense on smaller screens

## Suggested Build Order

1. Add A/B compare workflow
2. Add presentation mode
3. Expand pointer/touch interactivity
4. Add performance modes
5. Evaluate GIF export or richer recording presets

## Success Criteria

The roadmap is working if:

- users can compare and recover scenes without friction
- exported output is flexible enough for real sharing use cases
- the app works well for longer ambient or presentation sessions
- the product remains coherent across desktop and mobile
